# App-beskrivelse

OpgaveApp er en Android-app, der samler opgaver, vaner og økonomi i én løsning.
Formålet er at hjælpe brugeren med at være mere produktiv og motiveret ved at belønne færdige opgaver og daglige vaner med virtuelle penge. Appen giver samtidig overblik over saldo og historik over køb og belønninger.

# Feature-liste

Opret, færdiggør og slet opgaver

Automatisk dato på opgaver

Belønning: +10 kr for hver færdig opgave

Opret og følg vaner med streak

Daglig afkrydsning af vaner (+2 kr)

Kun én registrering pr. vane pr. dag

Saldo med oversigt over indtægter og udgifter

Registrering af køb/udgifter

Transaktionshistorik med farver (grøn/rød)

Data gemmes lokalt med SharedPreferences + Gson

Automatisk indlæsning af data ved opstart

Hvis du vil, kan jeg også tilføje en installationssektion, screenshots-afsnit eller en kort teknisk beskrivelse.
